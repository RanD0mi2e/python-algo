# LeetCode算法练习项目

这是一个用于练习LeetCode算法题的Python项目，提供了完整的开发和调试环境。

## 项目结构

```
├── src/                # 源代码目录
│   ├── solutions/      # 算法题解目录
│   └── utils/          # 工具类目录
├── tests/              # 测试用例目录
├── requirements.txt    # 项目依赖
└── README.md          # 项目说明
```

## 环境配置

1. 安装依赖包：
```bash
pip install -r requirements.txt
```

2. 运行测试：
```bash
pytest
```

## 使用方法

1. 在`src/solutions`目录下创建新的Python文件来实现算法
2. 在`tests`目录下创建对应的测试文件
3. 使用VSCode的调试功能进行代码调试
4. 运行`pytest`命令执行测试用例
